# The 5 Ingredients of a Good Design Prompt
1. Style Reference — name a movement or era

  Swiss/International style, Bauhaus, Brutalist, Memphis, Y2K, Cyberpunk, Art Deco

2. Color Instruction — be specific
  
  "monochrome only", "earth tones", "neon on dark background", "2 colors max"

3. Typography hint

 "bold display font", "serif headline", "all caps", "handwritten feel"

4. Mood/Feeling

 aggressive, calm, luxurious, playful, urgent, mysterious, corporate

5. What to AVOID — this is underrated

 "no gradients", "no clipart", "no busy backgrounds", "no AI faces"
